Проект Микросервиса по получению данных погоды из Яндекс-погоды, интегрированный с инструментами мониторинга Prometheus и
визуализации Grafana.

Инструкция по сборке и запуску приложения

Для сборки приложения необходимо установить: 1)Java Development Kit 2) Gradle 3) MongoDb Compass

Для полноценного функционирования java-приложений необходимо указать системе пути к каждому из установленных компонентов.
Порты и хост по умолчанию для всех контенеров указаны в файле docker-compose


Запуск можно произвести разными способами:

Используя IDE (на примере IntellijIdea) Выбрать в меню Run-> run Application

Для сборки и запуска docker-контейнеров:
Из командной консоли в папке приложения сгенерировать jar-файлы:
Зайдя в каталог модуля api запустить следующую команду в консоли: 
docker build -t api-service:latest .
Затем в каталоге модуля domain запустить следующую команду в консоли:
docker build -t weather-service:latest .

После этого поднимаем контейнеры в докере из основного каталога командой:
docker compose up
